---
title: How to Install NVM on Debian 12
date: 2024-12-21T10:05:00+08:00
---
```
sudo apt update 
sudo apt upgrade 
sudo apt install build-essential libssl-dev 
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/master/install.sh | bash 
source ~/.bashrc
command -v nvm 
nvm install 18 
nvm use 18 
```
